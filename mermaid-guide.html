Mermaid Diagram Guide

Create professional diagrams with simple text

Flowcharts | Process Maps | Org Charts | Timelines

What is Mermaid?

Mermaid is a way to create diagrams using simple text. Instead of dragging boxes around in PowerPoint or Visio, you describe the diagram and it renders automatically.

Why use it? You can ask ChatGPT to create Mermaid diagrams for you. Just describe what you want, and it will generate the code. Copy the code into a Mermaid viewer and you have a professional diagram.

Where to Use Mermaid

  • ChatGPT: Ask it to create a Mermaid diagram, then copy to a viewer
  • Mermaid Live Editor: mermaid.live - paste code, see diagram, export as PNG
  • Notion: Use code block with "mermaid" language
  • GitHub/GitLab: Renders in markdown files
  • VS Code: With Mermaid extension

Quick Start: Let AI Create Diagrams for You

The Easy Way: Just ask ChatGPT to create the diagram for you. You don't need to learn the syntax.

Example Prompt for ChatGPT

Create a Mermaid flowchart showing our customer onboarding process: 1. Customer signs contract 2. Account manager creates customer in system 3. IT sets up access credentials 4. Warehouse receives notification 5. Customer training scheduled 6. Go live

ChatGPT will generate something like:

flowchart TD A[Customer Signs Contract] --> B[AM Creates Customer in System] B --> C[IT Sets Up Credentials] C --> D[Warehouse Notified] D --> E[Training Scheduled] E --> F[Go Live]

Which renders as:

flowchart TD A[Customer Signs Contract] --> B[AM Creates Customer in System] B --> C[IT Sets Up Credentials] C --> D[Warehouse Notified] D --> E[Training Scheduled] E --> F[Go Live]
Pro Tip: Copy the Mermaid code from ChatGPT, paste it into mermaid.live, then click "Actions" → "Export PNG" to save the image for your presentation.

Flowcharts (Most Common)

Basic Flowchart

Code:

flowchart TD A[Start] --> B{Decision?} B -->|Yes| C[Do This] B -->|No| D[Do That] C --> E[End] D --> E

Result:

flowchart TD A[Start] --> B{Decision?} B -->|Yes| C[Do This] B -->|No| D[Do That] C --> E[End] D --> E

Shape Reference

Shape Code Use For
Rectangle [Text] Process steps, actions
Diamond {Text} Decisions, yes/no questions
Rounded (Text) Start/end points
Circle ((Text)) Connectors
Database [(Text)] Data stores, systems

Direction Options

Code Direction
flowchart TD Top to Down (vertical)
flowchart LR Left to Right (horizontal)
flowchart BT Bottom to Top
flowchart RL Right to Left

Tips & Tricks

Line breaks in text: Use <br> for line breaks inside boxes. Example: [First Line<br>Second Line]
Labeled arrows: Use -->|Label| to add text on arrows. Example: A -->|Yes| B
Dotted lines: Use -.-> for dotted arrows, good for optional or indirect relationships.
Subgraphs: Group related items with subgraph Name ... end
Just Ask AI: The easiest approach is to describe what you want to ChatGPT:

"Create a Mermaid flowchart showing [your process]. Include decision points for [X] and [Y]. Use subgraphs to group [these items]."
