AI Tool Guides

Comparison and quick-start guides for AI platforms

ChatGPT | Microsoft Copilot | Google Gemini | NotebookLM | Perplexity | Claude

Pro Tip: Use Voice Input

You can speak faster than you type. All major AI tools support voice input — use the microphone icon or your device's dictation feature. This is especially helpful for longer prompts and brainstorming sessions.

Note: All AI models can hallucinate (make up facts). Always verify important information.

ChatGPT (OpenAI)

ChatGPT

Free tier Plus: $20/mo

chat.openai.com

The most widely-used AI assistant. Excellent for general tasks, writing, coding, and analysis. GPT-4 (Plus) is significantly more capable than the free GPT-3.5.

Strengths

  • Most widely used AI assistant
  • Projects feature for organizing work
  • Custom GPTs for specific use cases
  • Image generation built-in
  • Mobile app with voice input

Limitations

  • Free tier is limited
  • Knowledge cutoff (not always current)
  • Rate limits on free tier
Best for Walker SCM: Day-to-day writing tasks, email drafting, document creation, brainstorming, explaining complex topics, Excel formulas.

Quick Start

  1. Go to chat.openai.com
  2. Sign up with email or Google account
  3. Start with: "Help me write a professional email about [topic]"
  4. Use the paperclip icon to upload files for analysis

Microsoft Copilot

Microsoft Copilot

Free tier M365 Copilot: Enterprise

copilot.microsoft.com

Microsoft's AI assistant, integrated with Bing search and Office 365. The free version is solid; the enterprise version integrates directly into Word, Excel, Outlook, and Teams.

Strengths

  • Built-in web search with citations
  • Enterprise version in Office apps
  • Image generation included
  • Good for Microsoft ecosystem users
  • Enterprise data protection

Limitations

  • Free version less capable than ChatGPT Plus
  • Enterprise version requires M365 license
  • Can be slower than competitors
  • Interface less intuitive
Best for Walker SCM: Already in our ecosystem. Good for quick web searches with sources, and if we deploy M365 Copilot, it will work directly in Outlook, Word, Excel, and Teams.

Quick Start

  1. Go to copilot.microsoft.com
  2. Sign in with your Microsoft account
  3. Try: "Search for recent trends in warehouse automation"
  4. Note the citations - you can verify sources

Google Gemini

Google Gemini

Free tier Advanced: $20/mo

gemini.google.com

Google's AI assistant. Full suite integration with Google Workspace — Docs, Sheets, Gmail, and video. Strong image capabilities and real-time information access.

Strengths

  • Full Google Workspace suite (Docs, Gmail, Sheets)
  • Strong at research tasks
  • Video and image capabilities
  • Real-time information access
  • Image generation built-in

Limitations

  • Less mature than ChatGPT
  • Sometimes less nuanced responses
  • Google account required
Best for Walker SCM: Research tasks, Google Workspace users, tasks requiring up-to-date web information, image analysis.

Quick Start

  1. Go to gemini.google.com
  2. Sign in with Google account
  3. Try: "Analyze this image" and upload a photo
  4. Or: "Help me research [logistics topic]"

NotebookLM (Google)

NotebookLM

Free

notebooklm.google.com

Google's document analysis tool. Upload PDFs, docs, or websites and ask questions about them. It ONLY answers from your uploaded sources - no hallucinations from external knowledge.

Strengths

  • Grounded in YOUR documents only
  • Cites exact sources from uploads
  • Great for policy/contract analysis
  • Can generate Audio Overviews, notes, PowerPoints
  • Completely free

Limitations

  • Only knows what you upload
  • No web search
  • Limited to ~50 sources per notebook
  • Requires Google account
Best for Walker SCM: Analyzing contracts, policies, RFPs, training materials, or any situation where you need answers grounded in specific documents without risk of hallucination.

Quick Start

  1. Go to notebooklm.google.com
  2. Create a new notebook
  3. Upload PDFs, docs, or paste website URLs
  4. Ask: "Summarize the key points" or "What does this say about [topic]?"
  5. Click "Audio Overview" to generate a podcast-style summary

Perplexity AI

Perplexity

Free tier Pro: $20/mo

perplexity.ai

AI-powered search engine that provides answers with citations. Think of it as "Google that gives you the answer instead of links." Every response includes numbered sources you can verify.

Strengths

  • Always cites sources
  • Real-time web search
  • Great for research and fact-finding
  • "Focus" modes for specific searches
  • Clean, simple interface

Limitations

  • Less capable for creative tasks
  • Not ideal for long-form writing
  • Free tier has usage limits
  • Less customizable than ChatGPT
Best for Walker SCM: Research with verifiable sources, competitor analysis, industry trends, any time you need to cite where information came from.

Quick Start

  1. Go to perplexity.ai
  2. No account needed for basic use
  3. Try: "What are the top trends in 3PL logistics for 2026?"
  4. Note the numbered citations - click to verify

Claude (Anthropic)

Claude

Free tier Pro: $20/mo

claude.ai

Anthropic's AI assistant. Known for strong reasoning, long document handling, and nuanced writing. Claude can process very long documents (up to 200k tokens - roughly a 500-page book). Claude Code is considered one of the best coding tools available.

Strengths

  • Excellent at long documents (500+ pages)
  • Strong analytical reasoning
  • Best-in-class coding with Claude Code
  • MCP ecosystem for integrations
  • Nuanced, thoughtful responses

Limitations

  • Free tier has message limits
  • Sometimes overly cautious
Best for Walker SCM: Analyzing long contracts or documents, coding tasks, nuanced writing, complex analysis that requires careful reasoning.

Quick Start

  1. Go to claude.ai
  2. Sign up with email
  3. Upload a long document and ask questions about it
  4. Try: "Analyze this contract and identify potential risks"

Which Tool Should I Use?

If you need to... Use this
Write an email or document ChatGPT, Claude, or Gemini
Research with citations Perplexity
Analyze a specific document NotebookLM (won't hallucinate)
Get help in Office apps Microsoft Copilot
Analyze a very long document Claude, Gemini, or ChatGPT
Create or understand code/formulas Claude or ChatGPT
Get current information Perplexity
Work with images Gemini or ChatGPT
